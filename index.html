Chapter 01 · Definition

コミュニケーションデザインとは、
未来の資産を設計する機能である。

目の前のプロジェクトを、単発の告知で終わらせずに、ファンとの関係・発信品質・判断基準という未来資産に変える。それが私たちのスコープだ。

— Definition

同じ接点でも、設計次第で残るものが変わる。
私たちは、案件をこなす仕事ではなく、案件を通じてクラブの未来資産を積み上げる仕事をしている。

01 / Context

前提

ファン・エンゲージメント部門が、チケット・イベント・ファンクラブを通じて、目の前の成果を担う。

02 / Role

Chapter 02 · Why Now

短期販促の延長線では、
クラブの未来は積み上がらない。

告知や販促は、出した瞬間に成果が出て、案件と一緒に消えていく。私たちが残すべきものは、その先にある「次もここに来たい理由」だ。

Before

“プロモーション”という言葉が、引き寄せる景色

  • 告知・販促・最終プッシュ
  • 短期成果の積み重ね
  • 案件ごとに終わってしまう
  • 担当者の感覚に依存する
After

コミュニケーションデザインとして、持ちたい視点

  • ファンとの関係づくり
  • 導線と体験の設計
  • 発信品質・トンマナの整備
  • 判断基準の言語化
  • 知見の蓄積と再利用
ファン・エンゲージメント部門が “いまの成果” を担うからこそ、
こちらは “未来に向けて貯める” 視点を持つ。

Chapter 05 · What We Do

接点・言葉・導線・品質・学びを、
設計する。

案件をこなすのではなく、案件を通じて「型」を残す。

01Relation

ファンとの関係を設計する

一度の来場・参加を、次の関係につなげる。

02Learning

成果と学びの両方で見る

売れたかだけでなく、何が効いたかまで残す。

03Quality

発信品質をコントロールする

誰に、何を、どう届けるかを整える。

04Language

感覚を言語化する

トンマナ・見せ方・判断理由を共有資産にする。

05Challenge

チャレンジを設計する

目的・仮説・検証を持った挑戦にする。

06Internal

社内の依頼精度を上げる

“出してください” を “何を達成したいか” に変える。

コミュニケーションデザインは告知業務ではなく、未来に残る設計業務である。
Chapter 06 · Assetization

写真単体ではなく、
“どう見せ、どう伝え、何を守るか” を残す。

ファンとの接点設計

接点の目的と体験設計を定義し、一貫したブランド体験を生む。

コピー・言い回し

理念や価値観を言語化し、共感を生むキーフレーズを蓄積する。

ビジュアル・トンマナ

色・質感・余白、統一感ある表現ルールを明文化する。

媒体ごとの役割

媒体特性に応じた役割を定義し、情報の使い分けを最適化する。

コミュニケーション資産を
生み出すサイクル

発信依頼フォーマット

依頼の目的・文脈・トーンを統一し、関係者が迷わず動ける形にする。

振り返りテンプレート

KPIだけでなく “学び” を記録し、改善につながる振り返りを習慣化する。

表現判断ログ

表現の意図と判断を記録し、迷いを減らし一貫性を担保する。

成功・失敗事例

ベスト／ワーストから再現性を抽出し、次の企画に使う。

担当者の感覚を、クラブとして 再利用できる判断基準と発信品質 へ変えていく。
Chapter 07 · Goals Catalogue

あなたが、この1年で
残すもの を選ぶ。

個人目標は「与えられるもの」ではなく、「選び取るもの」。4つのテーマごとに、ありえる目標の選択肢を用意した。自分の強み・興味・今期の挑戦領域に合わせて、2〜3つ選んで組み立てる。

i

カタログの読み方

各テーマには 4つのオプションがある。Level は背伸びの目安（1=確実 / 5=挑戦）。 定量・定性の指標は、上長と合意するときの起点として使う。 自分の今期の挑戦量と相談して、テーマごとに 1つ選び、合計 2〜3つ をマイ目標とする。

Theme 01

重点プロジェクトのコミュニケーション設計

目的・対象者・訴求・媒体・導線を整理し、行動につながる設計を行う。

4 Options
Option A · Match Lv

重点試合の集客設計を、1本リードする

注目試合（ダービー・ACL・シーズン終盤の山場など）から1つ選び、対象者・訴求・媒体・導線まで一気通貫で設計する。

Quant
設計案件 1〜2本 / 振り返り実施 100%
Qual
誰に何を届けるかが、資料1枚で説明できる
Option B · Annual Lv

年間プロジェクトの導線を、ゼロから組み立て直す

ファンクラブ・スポンサー連動・地域連携など、複数月にまたがる案件で「いつ・誰に・どの順で届けるか」を設計する。

Quant
導線設計 1本 / 月次レビュー 12回
Qual
全体像が一覧で見える状態をつくる
Option C · Campaign Lv

短期キャンペーンを、“設計案件” に格上げする

これまで告知ベースだったキャンペーンを、目的・KPI・導線が明文化された案件に変える。3〜5本実施。

Quant
格上げ案件 3〜5本 / 設計書化 100%
Qual
「なぜやるか」が誰でも答えられる
Option D · New Audience Lv

新規ファン獲得の導線を、ゼロから設計する

初観戦・初回来場をターゲットにした接点設計。認知→興味→来場→再訪までの導線を、1セット完成させる。

Quant
導線 1本完成 / 初観戦アンケート設計
Qual
初観戦の体験ストーリーが描ける
Theme 02

ファンとの関係づくりにつながる接点設計

初回来場や継続接点につながる設計をつくる。1回の接点で終わらせない。

4 Options
Option A · Segment Lv

セグメント別の訴求パターンを、3つ以上構築する

新規／ライト／コア／離脱層など、対象別に「どう声をかけるか」のテンプレを作り、実案件で運用する。

Quant
訴求パターン 3種以上 / 実運用 5案件以上
Qual
「誰に」が会議で迷子にならない
Option B · Next Touch Lv

“次の接点” を、すべての企画にデフォルトで組み込む

担当する案件すべてに「来場後、次にどこで会うか」を設計段階で組み込む。1回で終わる接点をなくす。

Quant
次回接点設計 担当案件の 100%
Qual
「次は◯◯で会える」が見える
Option C · Voice Lv

ファンの声を集める仕組みを、社内に1本立てる

来場後アンケート、SNS反応分析、座談会など、定常的にファンの声が入ってくる仕組みを設計・運用する。

Quant
仕組み 1本構築 / 月次レポート 6回以上
Qual
意思決定に「声」が入る
Option D · Dashboard Lv

ファンの熱量を可視化するダッシュボードをつくる

来場・購買・SNSなどのデータを統合し、ファン層ごとの状態が一目で見えるダッシュボードを構築する。

Quant
ダッシュボード 1本 / 月次更新 100%
Qual
マーケ会議が「数字 + 物語」で進む
Theme 03

発信品質・判断基準の言語化と資産化

表現判断とトンマナを記録し、次回に使える形で残す。属人化を、設計に変える。

4 Options
Option A · Tonality Lv

トンマナガイドを完成させ、社内に浸透させる

ビジュアル・コピー・SNS投稿の判断基準を1冊にまとめ、社内勉強会で共有・運用する。

Quant
ガイド完成 1本 / 社内勉強会 2回以上
Qual
新メンバーに渡せる状態になる
Option B · Decision Log Lv

表現判断ログを、月5件、書き続ける

「なぜこの表現にしたか」を、案件ごとに1件以上記録。1年で60件のクラブ判断資産をつくる。

Quant
判断ログ 月5件 × 12ヶ月 = 60件
Qual
過去判断を、次回に引用できる
Option C · Request Form Lv

発信依頼フォーマットを、標準化する

他部署からの依頼を「目的・対象・トーン」が必ず入る形にする。テンプレを作り、運用させる。

Quant
テンプレ運用 全依頼の 80% 以上
Qual
「出してください」が消える
Option D · Annual Report Lv

年次レポートで、設計知見を1冊にまとめる

1年の主要案件を「目的→設計→結果→学び」のフォーマットで整理し、社内資産として残す。

Quant
レポート 1冊 / 主要案件カバー率 80% 以上
Qual
来年の判断に、今年が使える
Theme 04

チャレンジを設計し、学びに変える

新しい訴求や見せ方を、仮説と検証を持って試す。「やってみた」で終わらせない。

4 Options
Option A · Format Lv

新フォーマットの発信を、仮説検証で3本以上試す

縦動画・長文ストーリー・インタビュー連載など、新しい発信形式を仮説→実行→振り返りで回す。

Quant
新フォーマット 3本以上 / 検証記録 100%
Qual
何が効いたかを、次に活かせる
Option B · AI / Tools Lv

AI・新ツールを、業務に1つ以上組み込む

コンテンツ生成補助・データ分析・社内ツールなど、新しいテクノロジーを実務に統合する。

Quant
ツール導入 1本 / 効果検証 1サイクル
Qual
「使い方」を社内に共有できる
Option C · Benchmark Lv

他クラブ・他競技のベンチマーク調査をリードする

国内外のクラブ・他競技・他業界から、自クラブに転用できる事例を月1本ペースで紹介・提案する。

Quant
事例紹介 月1本 × 12回 / 提案化 3本以上
Qual
外の視点が、社内に持ち込まれる
Option D · Launch Lv

新規プロジェクトを、自分の発案で立ち上げる

現場で感じた課題から、新しい接点・コンテンツ・取り組みを企画し、立ち上げまで持っていく。

Quant
新規企画 1本立ち上げ / 効果検証 完了
Qual
クラブの選択肢を増やせる
Chapter 08 · Team Goals & Evaluation

数字だけでも、理念だけでも弱い。
何を達成し、どんな状態をつくり、何を残したかで見る。

部目標（3本柱）

01

ファンベースの拡大・定着

短期施策を、継続的な関係づくりにつなげる。

02

短期プロジェクトの成果創出と学習化

成果を出しながら、何が効いたかを残す。

03

発信品質・判断基準の言語化と仕組み化

クラブらしい発信を再現できる状態をつくる。

評価軸

Quantitative

  • 案件数
  • 成果指標
  • 振り返り実施数
  • 資産化件数
  • 共有回数

Qualitative

  • 再現性
  • 判断の質
  • クラブらしさ
  • 次回活用度
  • 社内への浸透
